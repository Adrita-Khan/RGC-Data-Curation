
# RGC Data Curation for Radio Galaxy Classifier

**RGC (Radha Gobinda Chandra)** is a Radio AGN Classifier named after Radha Gobinda Chandra (1878â€“1975), a pioneering Bangladeshi-Indian astronomer and variable star observer from the first half of the twentieth century.

## Overview

This repository documents the data curation process for the RGC project. The quality and structure of the training data are critical for developing an accurate classifier.
